# Step 7. 이벤트와 State

## 완료 결과

상세 화면에 좋아요 버튼을 추가하고, 버튼 클릭 횟수를 `useState`로 관리합니다.

## 변경 내용

- `PostDetailPage`에서 `useState`를 사용합니다.
- 버튼 `onClick` 이벤트를 추가합니다.
- 클릭할 때마다 화면의 숫자가 바뀌도록 합니다.

## 동작 원리

- 이벤트는 사용자의 행동에 반응하는 코드입니다.
- state는 컴포넌트가 기억하는 값입니다.
- 다음 값이 이전 state에 의존하면 함수형 updater를 사용합니다.
- state가 바뀌면 React가 화면을 다시 그립니다.

## 결과 확인

상세 페이지에서 Like 버튼을 누를 때 숫자가 증가하면 됩니다.
