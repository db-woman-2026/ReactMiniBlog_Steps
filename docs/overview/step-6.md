# Step 6. 동적 라우트와 상세 페이지

## 배울 내용

게시글 카드에서 상세 페이지로 이동하고, URL의 id로 게시글을 찾습니다.

## 주요 변경

- `/posts/:postId` 라우트를 추가합니다.
- `src/pages/PostDetailPage.jsx`를 추가합니다.
- `useParams`로 URL 값을 읽습니다.
- 게시글이 없을 때 간단한 안내 문구를 보여줍니다.

## 학습 포인트

- `:postId`는 URL의 변하는 부분입니다.
- `useParams`는 현재 URL의 동적 값을 읽는 Hook입니다.
- 목록과 상세 화면은 같은 데이터를 다르게 보여주는 예입니다.

## 확인 방법

게시글 제목을 클릭했을 때 `/posts/1` 같은 주소로 이동하고 상세 내용이 보이면 됩니다.
