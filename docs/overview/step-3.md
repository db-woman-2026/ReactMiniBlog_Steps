# Step 3. React Router 기초

## 완료 결과

React Router를 추가해 URL과 화면을 연결합니다.

## 변경 내용

- `react-router-dom`을 설치합니다.
- `BrowserRouter`, `Routes`, `Route`, `Link`를 사용합니다.
- `/`, `/about`, `/posts` 주소를 만듭니다.
- `AboutPage`, `PostsPage`를 추가합니다.

## 동작 원리

- Vite는 라우팅을 제공하지 않습니다.
- React에서는 라우팅 라이브러리를 직접 설치해 사용합니다.
- Next.js의 파일 기반 라우팅과 달리 React Router는 코드에서 라우트를 선언합니다.

## 결과 확인

상단 메뉴에서 Home, About, Posts를 눌렀을 때 주소와 화면이 함께 바뀌면 됩니다.
