# Step 13. 게시글 검색

## 이 단계의 목표

Posts 목록에서 keyword 입력값으로 게시글을 필터링합니다.

## 주요 변경

- `PostsPage`에 keyword state를 추가합니다.
- 제목과 내용에 keyword가 포함된 글만 보여줍니다.
- 검색 결과가 없을 때 안내 문구를 보여줍니다.

## 학습 포인트

- 검색어도 state로 관리할 수 있습니다.
- `filter`는 조건에 맞는 항목만 남기는 배열 메서드입니다.
- 서버 검색을 배우기 전에는 브라우저가 가진 배열을 기준으로 검색할 수 있습니다.

## 확인 방법

검색어를 입력했을 때 목록이 줄어들고, 없는 검색어에는 빈 결과 안내가 보이면 됩니다.
