# Step 5. 배열과 목록 렌더링

## 이 단계의 목표

게시글 배열을 `map`으로 반복해 여러 개의 `PostCard`를 화면에 출력합니다.

## 주요 변경

- `src/data/posts.js`의 샘플 게시글을 여러 개로 늘립니다.
- `PostsPage`에서 `posts.map(...)`을 사용합니다.
- 각 카드에 `key`를 지정합니다.

## 학습 포인트

- React에서 목록은 배열을 기준으로 렌더링합니다.
- `map`은 배열의 각 항목을 컴포넌트로 바꿀 때 자주 사용합니다.
- `key`는 React가 목록 항목을 구분하는 데 필요합니다.

## 확인 방법

Posts 화면에 여러 개의 게시글 카드가 반복 출력되면 됩니다.
